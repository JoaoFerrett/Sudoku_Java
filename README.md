# 🧩 Java Sudoku

## 🇧🇷 Português

### 📌 Sobre o projeto

Este é um jogo de Sudoku desenvolvido em Java, com interface gráfica utilizando Swing.

O projeto inclui:

* 🎲 Gerador automático de tabuleiros (utilizando backtracking)
* 🧠 Validação de jogadas em tempo real
* 🔒 Diferenciação entre células fixas e editáveis
* 🖥️ Interface gráfica customizada

---

### 🚀 Funcionalidades

* Geração de Sudoku válido e aleatório
* Sistema de dificuldade baseado na remoção de números
* Verificação de regras do Sudoku (linha, coluna e bloco 3x3)
* Interface interativa para o usuário

---

### 🛠️ Tecnologias utilizadas

* Java
* Swing (Javax)
* Programação orientada a objetos

---

### ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/JoaoFerrett/Sudoku_Java.git
```

2. Entre na pasta do projeto:

```bash
cd Sudoku_Java
```

3. Compile e execute:

```bash
javac src/main/app/UIMain.java
java app.UIMain
```

---

### 📂 Estrutura do projeto

```
src/
 └── main/
     └── java/
         ├── app/        # Classe principal
```

## 🇺🇸 English

### 📌 About the project

This is a Sudoku game developed in Java, featuring a graphical interface built with Swing.

The project includes:

* 🎲 Automatic puzzle generator (using backtracking)

* 🔒 Fixed and editable cells
* 🖥️ Custom graphical interface

---

### 🚀 Features

* Random valid Sudoku generation
* Difficulty system based on number removal
